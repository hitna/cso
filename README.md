# slackのスタンプをカウントするアプリ
## 事前設定
* main.pyと同じフォルダにlocal.pyを作成する
* local.py内に定数を記述する
	```
	#投稿先のチャンネル名
	CHANNEL_NAME = "#random"
	#OAuth Access Token
	USER_TOKEN = "xoxp-xxxx-xxxx-xxxx-xxxx"
	#Bot User OAuth Access Token
	BOT_TOKEN = "xoxb-xxxx-xxxx-xxxx"
	```
## 実行方法
* pythonでmain.pyを実行してください。
	```
	python main.py
	```
